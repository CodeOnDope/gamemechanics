About Game Mechanics 333

Game Mechanics 333 is a course in the Game Studies and Design (B.S.) program at Old Dominion University in Virginia, USA. This Spring 2025 portfolio showcases prototypes developed by students in just 8 weeks under the instruction of Dineshkumar, Lecturer.

Students learn essential skills including 2D animation, physics simulation, user interface design, and game mechanics implementation. Each student develops a complete game prototype using Unity and C# programming.

These projects represent the application of core game design principles while exploring innovative mechanics within the shooting genre. All games are available for download.
